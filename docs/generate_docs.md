# Генерація документації за допомогою PHPDox

## Необхідні умови
- PHP >= 7.4
- PHPDox (рекомендується `phpdox.phar`)

## Інструкція

1. Додай PHPDoc-коментарі до всіх публічних класів, функцій і методів.
2. Переконайся, що в корені проєкту існує файл `phpdox.xml`.
3. Запусти команду:

```bash
php phpdox.phar
